Dynamic React Quiz Component
A small interactive quiz built with React and Typescript.

App features:

- Dinamic state management with useState;
- Conditional rendering of success/error feedback;
- Disabled submit button until a selection is made;
- Active highlighting of selected options;
- Blocking further changes after submission;
- Clean, reusable components (ListGoup, Alert, Button);
- Accessibility ready form;
- Typescript-typed props and state for safer code;
- Bootstrap integration for styling;

Shows the ability to handle user interactions, state, conditional UI updates and reusable components.
